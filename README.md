# Gokula Health – Smart Cattle Monitoring Application

## Project Overview
Gokula Health is an offline Android application developed using Kotlin and Android Studio to help farmers manage cattle records, monitor milk production, and track vaccination schedules.

## Features
- Cow Registration with photo and Ear Tag ID
- Daily morning and evening milk entry
- Monthly average milk yield calculation
- Milk trend visualization using MPAndroidChart
- Vaccination reminder notifications using AlarmManager
- Dashboard screen for cattle summary
- Offline functionality using SharedPreferences

## Technologies Used
- Kotlin
- Android Studio
- XML Layouts
- MPAndroidChart
- AlarmManager
- SharedPreferences

## How to Run
1. Open project in Android Studio
2. Sync Gradle files
3. Run using Android emulator
4. Use the app features

## Project Structure
- MainActivity.kt → Main backend logic
- DashboardActivity.kt → Dashboard summary
- SplashActivity.kt → Splash screen
- AlarmReceiver.kt → Vaccination reminder notifications
- activity_main.xml → Main frontend UI

## Conclusion
The application helps farmers digitally manage cattle and milk production efficiently through a simple offline Android system.
